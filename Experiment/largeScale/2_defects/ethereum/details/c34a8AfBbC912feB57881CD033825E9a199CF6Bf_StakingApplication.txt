Critical Variables Manipulation (CVM):
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of stake tokens in Function stake
Function setController is able to modify the value of Variable controller, which will influence the transfer of stake tokens in Function stake
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of reward tokens in Function claimRewards
Function setController is able to modify the value of Variable controller, which will influence the transfer of reward tokens in Function claimRewards
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of unstake tokens in Function unstake
Function setController is able to modify the value of Variable controller, which will influence the transfer of unstake tokens in Function unstake
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of unstake tokens in Function unstakeAll
Function setController is able to modify the value of Variable controller, which will influence the transfer of unstake tokens in Function unstakeAll
