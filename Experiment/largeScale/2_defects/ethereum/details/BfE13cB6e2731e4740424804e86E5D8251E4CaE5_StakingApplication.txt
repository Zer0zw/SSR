Critical Variables Manipulation (CVM):
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of stake tokens in Function stake
Function addVault is able to modify the value of Variable vaults, which will influence the transfer of stake tokens in Function stake
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of reward tokens in Function claimRewards
Function addVault is able to modify the value of Variable vaults, which will influence the transfer of reward tokens in Function claimRewards
Function setAccountFactory is able to modify the value of Variable accountFactory, which will influence the transfer of unstake tokens in Function unstake
Function addVault is able to modify the value of Variable vaults, which will influence the transfer of unstake tokens in Function unstake

Rewards without Timedelay (RT):
Function claimRewards calculate the reward without considering the time delay of staking
